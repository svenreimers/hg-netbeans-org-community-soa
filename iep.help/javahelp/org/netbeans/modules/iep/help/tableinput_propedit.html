Using the Table Input Operator Property Editor

See Also

The Table Input operator provides static event information to the IEP feature by providing a relational table as input. You can use the operator as table input to other downstream operators.

To create a Table Input operator:

  1. Drag a Table Input operator from the Palette to the Design view.
  2. Double-click the Table Input component. The property editor opens.
  3. Rename the component appropriately.
  4. Supply a value for Global ID, usually the same as the component name in the previous step.
  5. As needed, click Add Attribute and configure the input schema: Attribute Name, Data Type, Size, Scale, and Comment.
  6. To remove an unused attribute, select the attribute and click Delete.
  7. To move an attribute up or down, select the attribute and click Move Up or Move Down.
  8. (Optional) Click the Documentation tab and supply notes on this component.
  9. Click OK.
